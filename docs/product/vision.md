# Vision del producto

## Contexto

Aplicacion para tablet enfocada en personal sanitario para gestionar sesiones de evaluacion con escalas y pruebas.

## Alcance MVP

- Lista de sesiones.
- Crear sesion.
- Ejecucion de prueba.
- Pausa y reanudacion de sesion.
- Resumen final de sesion.
- Vista/modal de transcripcion de audio.

## No incluido en esta fase

- Autenticacion.
- Sincronizacion real.
- STT real.
- Logica clinica avanzada.

## Objetivo de UX

Flujo claro, rapido y robusto para uso en entorno clinico sobre tablet.
