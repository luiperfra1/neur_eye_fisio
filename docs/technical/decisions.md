# Technical decisions

## ADR-0001 Monorepo

Se adopta monorepo para mantener alineados mobile, backend y documentacion en una sola historia de cambios.

## ADR-0002 Mobile stack

React Native + TypeScript por productividad, ecosistema y mantenibilidad en app tablet.

## ADR-0003 Backend stack

Django + DRF por madurez, velocidad de desarrollo y claridad para APIs modulares.

## ADR-0004 Estrategia de crecimiento

Primero estructura y contratos; despues implementacion incremental de funcionalidad.
